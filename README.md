# Task Manager AI

O aplicație React + FastAPI pentru alocarea automată a taskurilor către angajați, cu UI responsive și backend REST.

## Tehnologii

- React (Vite)
- TailwindCSS
- FastAPI (Python)
- Vercel + Render

## Comenzi utile (local)

```bash
npm install
npm run dev
```

## Variabile de mediu

```
VITE_API_URL=https://[backend-taul-on-render].onrender.com
```
